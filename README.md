# Gewenbo Store

A storefront for premium bilingual educational resources.

[![Live Demo](https://img.shields.io/badge/demo-gewenbo--store.vercel.app-111)](https://gewenbo-store.vercel.app)

## Overview

A simple storefront for digital downloads and interactive guides — covering AI tools, history, technology, and philosophy — presented bilingually in English and Chinese.

## Design & Tech

- **Stack:** Single HTML file, vanilla CSS
- **Deployment:** Vercel
